**Questa è una scheda di esempio.** Serve a mostrare la struttura di una decisione
nella raccolta: qui va il commento libero in Markdown — sintesi dei fatti, passaggi
rilevanti della motivazione, eventuali note.

Per aggiungere una decisione vera:

1. copia questo file in `content/giurisprudenza/` con un nome parlante
   (es. `trib-roma-2026-06-20-frontiera.md`);
2. compila i campi in alto (corte, tipo, numero, date, temi, massima, norme);
3. se hai il PDF del provvedimento, mettilo in `public/allegati/giurisprudenza/`
   e indica il percorso nel campo `pdf`;
4. elimina questo file di esempio.
